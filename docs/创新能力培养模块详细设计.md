## 1. 功能概述

创新能力培养模块是Aigc For Study的核心差异化功能之一，通过AI驱动的创新训练体系，帮助学生提升批判性思维、创新思维和实践能力。

### 1.1 设计理念

- **AI引导创新**：利用AI生成开放性问题、项目建议，激发学生创新思维
- **实践驱动学习**：通过真实项目实践，将创新思维转化为实际能力
- **成果可视化**：展示学生创新成果，形成正向激励闭环
- **社交化创新**：支持创意挑战和成果分享，营造创新氛围

### 1.2 功能模块

| 模块 | 核心功能 | AI能力 | 优先级 |
|------|----------|--------|--------|
| 创新思维训练 | 开放性思考题、思维导图、批判性分析 | AI生成思考题、AI点评分析 | P0 |
| 项目实践 | 项目创建、进度管理、团队协作、成果提交 | AI项目建议、AI进度提醒 | P0 |
| 创意挑战 | 挑战赛发布、作品提交、投票评选 | AI挑战题生成、AI作品点评 | P1 |
| 创新成果展示 | 成果展示墙、点赞评论、荣誉体系 | AI成果推荐、AI评语生成 | P1 |

## 2. 创新思维训练

### 2.1 功能详情

| 功能点 | 描述 | 验收标准 |
|--------|------|----------|
| AI生成思考题 | 根据学习领域生成开放性思考题 | 每日生成3-5道思考题，覆盖不同思维维度 |
| 思维导图工具 | 学生可创建思维导图整理思路 | 支持节点添加、连线、导出 |
| 批判性分析 | 对观点进行多角度分析训练 | AI提供分析框架和点评 |
| 思维训练记录 | 记录学生的思维训练历程 | 可查看历史训练记录和成长曲线 |

### 2.2 思考题类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 发散思维 | 激发创意，多角度思考 | "如果手机没有屏幕，你会如何设计交互方式？" |
| 批判思维 | 分析论证，发现漏洞 | "分析'AI将取代所有人类工作'这一观点的合理性" |
| 逆向思维 | 反向思考，突破常规 | "如果要让用户讨厌一款产品，你会怎么做？" |
| 系统思维 | 整体视角，关联分析 | "分析外卖平台对城市交通的影响链" |
| 创新应用 | 知识迁移，创新实践 | "如何用数据结构知识优化外卖配送路径？" |

### 2.3 AI点评机制

```python
class ThinkingQuestionAI:
    def generate_question(self, domain: str, thinking_type: str) -> dict:
        """生成开放性思考题"""
        prompt = f"""
        请为{domain}领域生成一道{thinking_type}类型的开放性思考题。
        要求：
        1. 问题具有启发性，没有标准答案
        2. 能激发学生深入思考
        3. 与学生当前学习内容相关
        
        返回格式：
        {{
            "question": "问题内容",
            "hints": ["提示1", "提示2"],
            "evaluation_criteria": ["评价标准1", "评价标准2"]
        }}
        """
        return self.ai_client.generate(prompt)
    
    def evaluate_answer(self, question: str, answer: str) -> dict:
        """评价学生回答"""
        prompt = f"""
        请评价学生对以下思考题的回答：
        
        问题：{question}
        学生回答：{answer}
        
        从以下维度评价（1-10分）：
        1. 思考深度
        2. 逻辑性
        3. 创新性
        4. 表达清晰度
        
        并给出具体的改进建议。
        """
        return self.ai_client.generate(prompt)
```

## 3. 项目实践

### 3.1 功能详情

| 功能点 | 描述 | 验收标准 |
|--------|------|----------|
| 项目创建 | 学生可创建创新项目 | 支持项目名称、描述、目标、时间规划 |
| AI项目建议 | AI根据学生兴趣推荐项目方向 | 推荐3-5个相关项目建议 |
| 进度管理 | 跟踪项目进度，设置里程碑 | 支持甘特图视图、进度百分比 |
| 团队协作 | 支持组队完成项目 | 支持邀请成员、角色分配 |
| 成果提交 | 提交项目成果（文档、代码、演示） | 支持多种格式上传 |
| AI进度提醒 | AI根据进度自动提醒 | 临近截止日期自动提醒 |

### 3.2 项目类型

| 类型 | 说明 | 示例项目 |
|------|------|----------|
| 技术创新 | 开发新技术或改进现有技术 | "基于AI的智能学习助手" |
| 产品设计 | 设计创新产品或服务 | "校园共享雨伞系统设计" |
| 商业创新 | 商业模式创新或创业计划 | "校园二手交易平台商业模式" |
| 社会创新 | 解决社会问题的创新方案 | "城市垃圾分类智能解决方案" |
| 学术研究 | 创新性学术研究项目 | "AI在教育领域的应用研究" |

### 3.3 项目生命周期

```
创建项目 → AI建议优化 → 设置里程碑 → 执行开发 → 阶段性提交 → 最终成果 → 评价反馈
```


## 4. 创意挑战

### 4.1 功能详情

| 功能点 | 描述 | 验收标准 |
|--------|------|----------|
| 挑战赛发布 | 教师/平台发布创意挑战赛 | 支持主题、规则、时间、奖励设置 |
| AI挑战题生成 | AI生成创意挑战题目 | 题目具有启发性、可执行性 |
| 作品提交 | 学生提交挑战作品 | 支持多种格式（文档、图片、视频） |
| 投票评选 | 支持投票和专家评审 | 投票公平、结果透明 |
| AI作品点评 | AI自动点评参赛作品 | 点评具有建设性、针对性 |
| 奖励发放 | 发放虚拟奖励和荣誉 | 支持徽章、积分、证书 |

### 4.2 挑战赛类型

| 类型 | 周期 | 说明 |
|------|------|------|
| 每周微挑战 | 7天 | 小型创意任务，快速完成 |
| 月度挑战赛 | 30天 | 中型挑战，需要一定时间准备 |
| 学期大赛 | 一学期 | 大型创新项目，综合能力考验 |
| 专项挑战 | 不定 | 针对特定领域的创新挑战 |

### 4.3 数据模型

```sql
-- 创意挑战赛表
CREATE TABLE innovation_challenges (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    title VARCHAR(200) NOT NULL,
    description TEXT,
    challenge_type VARCHAR(50),  -- weekly/monthly/semester/special
    requirements TEXT,
    start_date TIMESTAMP,
    end_date TIMESTAMP,
    voting_end_date TIMESTAMP,
    rewards JSONB,  -- [{rank, reward}]
    status VARCHAR(20) DEFAULT 'upcoming',  -- upcoming/ongoing/voting/ended
    created_by UUID REFERENCES users(id),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 挑战作品表
CREATE TABLE challenge_submissions (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    challenge_id UUID NOT NULL REFERENCES innovation_challenges(id),
    user_id UUID NOT NULL REFERENCES users(id),
    title VARCHAR(200) NOT NULL,
    description TEXT,
    content_urls JSONB,  -- [{name, url, type}]
    vote_count INT DEFAULT 0,
    rank INT,
    ai_feedback TEXT,
    submitted_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 投票记录表
CREATE TABLE challenge_votes (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    challenge_id UUID NOT NULL REFERENCES innovation_challenges(id),
    submission_id UUID NOT NULL REFERENCES challenge_submissions(id),
    voter_id UUID NOT NULL REFERENCES users(id),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    UNIQUE(voter_id, challenge_id)  -- 每人每挑战只能投一票
);
```

## 5. 创新成果展示

### 5.1 功能详情

| 功能点 | 描述 | 验收标准 |
|--------|------|----------|
| 成果展示墙 | 展示学生创新项目成果 | 支持卡片式展示、分类筛选 |
| 点赞评论 | 其他用户可点赞和评论 | 支持点赞、评论、分享 |
| AI成果推荐 | AI推荐相关优秀成果 | 基于兴趣和领域推荐 |
| AI评语生成 | AI自动生成成果评语 | 评语具有鼓励性和建设性 |

| 项目先锋 | 完成3个创新项目 | 徽章 + 200积分 |
| 挑战冠军 | 获得挑战赛第一名 | 证书 + 500积分 |
| 创新大师 | 累计获得1000点赞 | 徽章 + 1000积分 |
